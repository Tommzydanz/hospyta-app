## Installation

1. Clone the repository:
    `git clone https://github.com/Tommzydanz/hospyta-app.git`
2. Navigate to the project directory:
    `cd hospyta-app`
3. Install dependencies:
    `yarn install`
4. Start the Expo development server:
    `npx expo start`
